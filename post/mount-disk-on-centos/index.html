挂载硬盘

查看新磁盘

查看硬盘

df -h

1. 查找未挂载的硬盘

[root@cgsl ]# fdisk –l

找到新添加的磁盘的编号为/dev/xvdb1

2. 创建硬盘分区

使用 fdisk /dev/xvdb1

输入n
Command (m for help):n

输入p

回车

回车

回车

输入w
Command (m for help): w

3. 格式化分区

# mkfs.ext4 /dev/xvdb1

4. 建立挂载目录

# mkdir /data

5. 挂载分区

# mount /dev/xvdb1 /data

6. 设置开机自动挂载

vi /etc/fstab

/dev/xvdb1 /data ext4 defaults 0 0

7. 确认是否挂载成功

重启服务器

# reboot
