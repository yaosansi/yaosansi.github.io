IIS7的备份与恢复

2008-01-12
  • web

在%systemdrive%\windows\system32\inetsrv\下应该有个工具–Appcmd.exe，我们将使用该工具进行IIS7的备份及恢复。

备份

AppCmd Add Backup “BackupName”

罗列备份

AppCmd List Backup

恢复

AppCmd Restore Backup “BackupName”
